WebNews
=======

WebNews is a simple RSS reader.

* JavaScript Enabled
* jQuery Ready
* Canvas with EaselJS ready
* Internal/External Browser navigation
* PhoneGap API access

...More features available soon...

Use the web as simply as you wish it to be.
